# ClutchMate Documentation

Welcome to the ClutchMate project documentation folder. This directory contains everything you need to understand, run, and present the app for the hackathon.

## Contents

- `README.md` — overview of the project and docs map
- `setup.md` — environment setup and run instructions
- `architecture.md` — system architecture, tech stack, and component design
- `api.md` — available backend endpoints and examples
- `hackathon-strategy.md` — pitch/feature guide, demo flow, and winning tactics

## Project Context

ClutchMate is a study companion app built with:
- **Backend**: Flask + Flask-SQLAlchemy + SQLite
- **Frontend**: static HTML/CSS/JS under `frontend`
- **Auth**: JWT-based signup/login
- **Key features**: personalized dashboard, study planning, assignments, wellness tracking, AI tutor-style helper, productivity analytics

## How to use this folder

1. Read `setup.md` to install dependencies and start the app.
2. Read `architecture.md` to understand the system design.
3. Read `api.md` if you need backend route references.
4. Use `hackathon-strategy.md` to shape your demo and pitch.
